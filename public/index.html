EggArena Web

Браузерная версия игры

Запускайте EggArena прямо из GitHub Pages как обычный игровой сайт. Прогресс сохраняется в браузере, а матч открывается по кнопке ниже.

Играть в браузере

Веб-режим запускается как локальная браузерная версия без установки приложения.

Сохранения Монеты, убийства и выбранные предметы хранятся в вашем браузере.
Управление После загрузки матча игрок появляется на карте, проходит отсчёт, и только затем включается движение.
Запуск с GitHub После деплоя откроется обычная ссылка на сайт, как у стандартной браузерной игры.
